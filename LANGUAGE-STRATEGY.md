## Принцип отбора / Selection Principle

17 языков корпуса выбраны по трём критериям, применённым одновременно:

1. **Статистика туристического потока** — реальные туристы в Узбекистан (Агентство статистики РУз, 2023–2024)
2. **Официальные языки ООН** — международная научная коммуникация
3. **Исламский и центральноазиатский контекст** — языки зиёрат-практики

## Стратегия перевода / Translation Strategy

### Принцип множественных эквивалентов
*(Mirror of babur-terra-codex approach)*

Каждый термин имеет **несколько эквивалентов** в каждом языке — потому что:
- Разные эквиваленты захватывают разные семантические нюансы
- «Правильного» единственного перевода для PLT-нагруженных терминов не существует
- Список эквивалентов = семантическая карта, а не словарная статья

Пример: **мазор** →  
- EN: `["mazar", "shrine", "saint's tomb", "sacred mausoleum"]`  
- Каждый вариант захватывает другой аспект: `mazar` = транслитерация, `shrine` = функция, `saint's tomb` = тип объекта, `sacred mausoleum` = архитектурная форма

### Транслитерация / Transliterations
Для языков с нелатинским / нелатинизированным письмом предоставляется:
- Исходный скрипт (кириллица, арабский, деванагари, ханчул, хань)
- Latin transliteration (`_latin` suffix in JSON)

### Этимологический блок / Etymology block
Каждый термин имеет `etymology` с:
- Арабским корнем (для исламских терминов)
- Тюркским корнем (для тюркских терминов)
- Путём вхождения в узбекский язык
- Семантическим сдвигом: исходное значение → специализированное зиёрат-значение
