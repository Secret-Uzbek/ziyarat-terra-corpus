## Что такое PLT в зиёрат-контексте

**PLT (Post-Lingua Trace)** — класс механизмов передачи значения, которые работают *ниже или за пределами* лингвистического кодирования: через воплощённую практику, пространственную навигацию, тактильный ритуал, материальную форму или молчание.

*(Abdukarimov A. Invisible Shadow. Zenodo, 2025)*

В зиёрат-практике PLT присутствует на **каждом уровне**:

## PLT-карта зиёрат-ритуала

### Уровень 1: Пространственный (Spatial PLT)

```
Зиёрат начинается ЗА пределами мазора:

[Дом] ──нийят──► [Маршрут] ──направление──► [Порог мазора]
         ↑                    ↑                      ↑
   Устное PLT         Пространственный PLT      Телесный PLT
   (формула)          (приближение с             (снятие обуви,
                       правильной стороны)        поклон)
```

Паломник не читает инструкцию. Он **знает тело** — снять обувь, склонить голову, войти с правой ноги. Это PLT: телесное знание без текста.

### Уровень 2: Тактильный (Tactile PLT)

| Действие | Объект | PLT-механизм |
|---|---|---|
| Прикосновение к порогу мазора | Камень / дерево | Баракá-перенос через тактильный контакт |
| Завязывание матоси | Решётка мазора | Материализация нийяти (намерения) |
| Поцелуй дарвозы | Ворота мазора | Субмиссия + баракá-активация |
| Прикосновение к ткани гробницы | Парча / сузани | Прямой баракá-канал к руҳи авлиё |
| Взятие тупроғи (земли мазора) | Почва | Материальный TraceLog-фрагмент |

**Корпусный вопрос**: Эти действия **не имеют словесного обозначения** в письменном корпусе. Они передаются показом, а не объяснением.

### Уровень 3: Акустический (Acoustic PLT)

Дуо (молитва) в зиёрат-контексте передаётся через **просодику, а не семантику**:

```
Содержание дуо: "О Боже, прими моё паломничество"
Механизм: мелодический контур + ритм + тональность
          ↑                    ↑           ↑
       PLT channel         PLT channel   PLT channel
       (не слова)          (не слова)    (не слова)
```

Оtin-oyi НЕ УЧИТ дуо через текст. Она произносит — ученица повторяет. Форма усваивается **акустически**. Это PLT.

### Уровень 4: Визуальный (Visual PLT)

- Цвет матоси на дереве: белый / зелёный / красный → нийят-кодирование без слов
- Количество свечей: число передаёт намерение → PLT числовой код
- Пространственное положение: женщины — отдельная зона → PLT гендерной сегрегации

### Уровень 5: Темпоральный (Temporal PLT)

Чилла (40 дней) — не текст. Это **длительность как PLT**:
- 40 = трансформационный цикл (библейское и доисламское подтверждение)
- Каждый из 40 дней = один шаг трансмиссии
- «Биз чилла пайтида ҳечнарса кўтармаймиз, фақат ниятимизни кўтарамиз» — NULLO+PLT в одной фразе

## Корпусологическая импликация

### Почему текстовый корпус недостаточен

Стандартный корпус (uzbekcorpora.uz) содержит **только Level 0 PLT** — тексты. Он **не содержит**:

| PLT-уровень | Что упущено |
|---|---|
| Spatial | Маршруты паломничества — данные GPS |
| Tactile | Описание тактильных ритуалов — только этнографические статьи |
| Acoustic | Аудио дуо / зикр / алла — не в текстовом корпусе |
| Visual | Материальная культура мазора — фотоданные |
| Temporal | Чилла-дневники — устная традиция |

### Предложение: PLT-корпус нового типа

Для полного исследования зиёрат-практики необходим **мультимодальный PLT-корпус**:

```
ziyarat-terra-corpus v2.0 (предложение):
├── text/          ← традиционный текстовый корпус
├── audio/         ← аудиозаписи дуо, зикр, алла, оtin-oyi speech
├── video/         ← видеодокументация ритуалов (аnonymised)
├── spatial/       ← GPS-маршруты паломничества (анонимизировано)
├── material/      ← фотодокументация PLT-объектов (матос, тумор, тупроқ)
└── temporal/      ← чилла-дневники (устные нарративы, транскрибированные)
```

Это принципиально новый тип корпуса для корпусной лингвистики.
