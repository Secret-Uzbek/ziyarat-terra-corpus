# Technical Glossary

This file defines a compact donor glossary for recurring Terra, FMP, AIUZ, and
publication-facing technical terms.

## Purpose

The glossary exists so repositories do not keep redefining the same technical
language from scratch.

It supports:

- readable public surfaces;
- stable cross-repository terminology;
- publication coherence;
- bibliography and citation repair;
- reduction of semantic drift.

## Core terms

### Codex

A norm-and-meaning layer that binds operational rules, trace discipline, and
human-facing interpretation.

### Trace

The continuity layer that preserves provenance, session memory, repair history,
and operational sequence.

### Resilience

The ability of a repository, workflow, or knowledge layer to survive noise,
damage, and fragmentation without losing recoverability.

### Layer

A distinct functional stratum within a repository or ecosystem, such as theory,
practice, governance, publication, machine, or archive.

### Protocol

A declared operational sequence or rule set that governs how work, validation,
or publication proceeds.

### Governance

The decision and responsibility layer that governs permissions, obligations,
roles, and escalation paths.

### Ontology

A structured meaning map defining entities, relations, roles, and conceptual
boundaries.

### Standard

A repeatable rule for format, structure, validation, or publication quality.

### DID

Decentralized identifier logic used for identity representation and traceable
agent or stakeholder roles.

### DAO role

A declared role within a governance structure, such as `user`, `operator`,
`stakeholder`, `verifier`, or `arbiter`.

### Semantic Integrity

A validation condition requiring that a change preserve meaning coherence rather
than only passing syntactic checks.

### Thesaurus

A semantic lexicon layer supporting multilingual term mapping, concept mapping,
and contextual interpretation.

### Session Continuity

The operational memory connecting one working session to the next without
forcing re-creation from zero.

### Multilingual Packaging Protocol

A rule that publication or repository artifacts be packaged across required
language surfaces with explicit structure and traceability.

## Use rule

If a repository uses these terms in a public or publication-facing way, it
should either:

- link back to this glossary;
- inherit its definitions directly;
- or document deliberate local divergence.
