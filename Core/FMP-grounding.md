## Четыре слоя FMP в зиёрат-дискурсе

### NULLO — Нолевая онтология

**Что это**: выживание традиции при нулевых институциональных ресурсах.

**В зиёрат-контексте**:
- Советский период (1927–1991): мечети закрыты, мазары официально запрещены
- Оtin-oyi продолжает работу: без зарплаты, без помещения, без разрешения
- Мазор-зиёрат не прекращается: женщины приходят «собирать травы», «кормить детей»
- 64-летнее выживание = задокументированный NULLO

**Корпусная метка**: `"fmp_layer": "NULLO"` в lexicon для отин-ойи, чилла

### PLT — Тилдан кейинги из

**Что это**: трансмиссия значения за пределами языковых структур.

**В зиёрат-контексте**:
- Тактильный ритуал у мазора
- Пространственный маршрут паломничества
- Просодика дуо (форма, не содержание)
- Молчание чиллы

**Корпусная метка**: `"plt_status"` в каждом термине

### UCOMM — Универсальная коммуникация

**Что это**: коммуникационный протокол между живыми и священным прошлым.

**В зиёрат-контексте**:
- Зиёрат = UCOMM-сессия: паломник ↔ авлиё ↔ Аллоҳ
- Баракá = UCOMM-сигнал (передаётся через прикосновение, пространство)
- Дуо = UCOMM-сообщение (формула активирует канал)
- Чилла = UCOMM-марафон (40 дней непрерывной сессии)

**Корпусная метка**: `"fmp_layer": "UCOMM"` для зиёрат, барака

### EUO — Самоорганизующаяся универсальная организация

**Что это**: самоорганизация без центральной власти.

**В зиёрат-контексте**:
- Тариқат-сеть: нет штата, нет правительства, работает 600 лет
- Хонақоҳ-сеть: финансируется вақфом, самовоспроизводится
- Зиёрат туризм 2026: государство формализует уже существующую самоорганизацию

**Корпусная метка**: `"fmp_layer": "EUO"` для хонақоҳ, тариқат, зиёрат туризми

### TraceLog — Узлы цивилизационной памяти

**Что это**: материальные узлы, в которых концентрируется «след» цивилизации.

**В зиёрат-контексте**:
- Мазор = TraceLog-узел (тело авлиё = физический якорь памяти)
- Мақбара = архитектурный TraceLog
- Вақф-документ = юридический TraceLog
- Сузани = текстильный TraceLog (узор = PLT)

**Корпусная метка**: `"fmp_layer": "TraceLog"` для мазор, мақбара, вақф
